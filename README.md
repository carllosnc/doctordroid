# Doctor Droid

**Doctor Droid** is a comprehensive Android diagnostic and performance monitoring application built with Jetpack Compose. It provides real-time insights into your device's hardware and system status through a modern, clean interface.

## Features

- **Device Information**: Detailed hardware specs (Manufacturer, Model, Board, Android version, etc.).
- **Storage Monitoring**: Visual breakdown of internal storage usage by category.
- **Battery Health**: Real-time level, status, health, temperature, and voltage with dynamic charging animations.
- **CPU & Performance**: Monitor processor load, core activity, and architecture.
- **Network & Connectivity**: Track WiFi and Cellular status, IP addresses, and signal details.
- **Audio & Sound**: Manage volume levels and check audio feature support.
- **Quick Controls**: Access screen brightness and volume directly within the app.
- **Floating Monitor**: Real-time overlay to track system performance while using other apps.

## UI/UX Design

- **Modern Blue Palette**: Built using a custom Tailwind-inspired blue color theme.
- **Material 3**: Fully compliant with Material Design 3 standards.
- **Minimalist Layout**: Uses a clean, card-based list design with subtle dividers.
- **Responsive Animations**: Includes smooth donut charts and charging pulse effects.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material 3
- **Navigation**: Compose Navigation
- **Architecture**: Modern Android development practices (Clean, readable components).

## Getting Started

1. Clone the repository.
2. Open the project in **Android Studio**.
3. Build and run the `:app` module on your Android device or emulator.
